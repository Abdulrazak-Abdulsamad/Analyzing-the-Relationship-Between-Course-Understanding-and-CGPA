# Analyzing-the-Relationship-Between-Course-Understanding-and-CGPA

This repository contains an R-based data analysis project that investigates the relationship between students' level of understanding of a course and their academic performance (CGPA). The project demonstrates the complete data analysis workflow, including data cleaning, exploratory data analysis, visualization, statistical hypothesis testing, and interpretation of results.

## Features

* Data cleaning and preprocessing using the tidyverse
* Standardization of categorical variables
* Boxplots and bar charts for visualizing CGPA across understanding levels
* Descriptive summary statistics
* One-Way ANOVA to test for significant differences in CGPA
* Tukey HSD post-hoc analysis for pairwise group comparisons
* Automated interpretation of statistical findings

## Technologies Used

* R
* tidyverse
* ggplot2
* janitor

## Learning Outcomes

This assignment demonstrates practical skills in:

* Data cleaning and transformation
* Exploratory Data Analysis (EDA)
* Statistical hypothesis testing
* Data visualization
* Communicating insights from real-world educational data

This project was completed as part of a Data Science assignment using R to apply statistical techniques for educational data analysis.
